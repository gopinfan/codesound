# 安装 Homestead

Homestead 管理脚本

https://github.com/laravel/homestead

Homestead 虚拟机盒子

https://app.vagrantup.com/laravel/boxes/homestead

下载虚拟机盒子

Laravel China 定制版 Homestead 虚拟机盒子

解压后包含两个文件：

  • virtualbox.box 定制版 Homestead 盒子
  • metadata.json 盒子的导入配置文件

导入盒子

> vagrant box add metadata.json

下载定制版 Homestead 管理脚本：

> cd ~
> git clone https://git.coding.net/summerblue/homestead.git Homestead

检出指定版本的脚本：

> cd ~/Homestead
> git checkout v7.8.0

初始化 Homestead

> bash init.sh

# Homestead 配置

Homestead 配置文件位于 ~/Homestead/Homestead.yaml

---
ip: "192.168.10.10"
memory: 2048
cpus: 1
provider: virtualbox

authorize: ~/.ssh/id_rsa.pub

keys:
    - ~/.ssh/id_rsa
    - ~/.ssh/id_rsa.pub

folders:
    - map: ~/Code
      to: /home/vagrant/Code

sites:
    - map: laravel.test
      to: /home/vagrant/Code/Laravel/public

databases:
    - laravel

# 虚拟机配置

默认配置，无需修改

ip: "192.168.10.10"
memory: 2048
cpus: 1
provider: virtualbox

# SSH 密钥登录配置

虚拟机授权连接公钥：

authorize: ~/.ssh/id_rsa.pub

同步本机公钥和私钥到虚拟机：

keys:
    - ~/.ssh/id_rsa
    - ~/.ssh/id_rsa.pub

检查是否存在 SSH Key:

> ls -al ~/.ssh

如果不存在 id_rsa 和 id_rsa.pub，使用以下命令生成 SSH Key:

> ssh-keygen -t rsa -C "your-email@exmple.com"

# 共享文件夹配置

folders:
    - map: ~/Code
      to: /home/vagrant/Code
  • map 本机文件夹
  • to 对应的 Homestead 上的文件夹

# 站点配置

sites:
    - map: laravel.test
      to: /home/vagrant/Code/Laravel/public

# 数据库配置

指定需要创建的数据库名称：

databases:
    - laravel

# 绑定 hosts

打开 hosts 文件

  • Mac 系统在 /etc/hosts
  • Windows 系统在 C:/Windows/System32/Drivers/etc/hosts

在最后添加

192.168.10.10  laravel.test

创建项目 Laravel 后，可通过浏览器访问 laravel.test 站点。

# 运行 Vagrant

启动 Homestead :

> cd ~/Homestead
> vagrant up

登录虚拟机：

> vagrant ssh

退出虚拟机：

$ exit

关闭 Homestead

> vagrant halt

# Vagrant 常用命令

Vagrant 的常用命令如下：

命令 说明
vagrant init 初始化 vagrant
vagrant up 启动 vagrant
vagrant halt 关闭 vagrant
vagrant ssh 通过 SSH 登录 vagrant（需要先启动 vagrant）
vagrant provision 重新应用更改 vagrant 配置
vagrant destroy 删除 vagrant
